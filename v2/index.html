Introduction

This page explains how to use the n-points / m-mirrors version.

Prerequisite

The current implementation depends on EPnP provided by EPFL. Please download the matlab code and extract them in ./matlab/ directory. Our code assumes ./matlab/efficient_pnp_gauss.m is available.

Usage

The usage is essentially identical to the 3-points / 3-mirrors version. See matlab/demo.m to change the number of points and mirrors.

Limitations

C++ version is not available for now.
